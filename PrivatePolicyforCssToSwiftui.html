Privacy Policy for CssToSwiftui

Effective Date: March 13, 2026

CssToSwiftui respects your privacy. This Privacy Policy explains what information the extension accesses, how it is used, and whether it is shared or stored.

1. What the extension does

CssToSwiftui allows users to select elements on a web page and convert their visual styles and structure into SwiftUI code for development and design reference.

2. Information the extension accesses

To provide its functionality, the extension may access:

  • Web page content, including selected page elements, their DOM structure, text content, and computed CSS styles
  • User interaction on the current page, such as clicks, hover actions, and element selection related to using the extension
  • Local extension state, such as activation status and limited usage-related data stored in the browser

3. How this information is used

The accessed information is used only to:

  • Let the user select and inspect elements on the current web page
  • Generate SwiftUI code based on the selected element’s structure and styling
  • Maintain basic extension functionality, such as activation state and local usage settings

4. Data sharing

CssToSwiftui does not sell user data.

CssToSwiftui does not share web page content, user interaction data, or personal information with third parties for advertising, marketing, or analytics purposes.

5. Data storage

The extension may store limited data locally in the user’s browser using Chrome extension storage, such as activation status and usage-related settings.

Web page content selected for conversion is processed locally in the browser and is not transmitted to our servers.

6. Remote code

CssToSwiftui does not use remotely hosted code. All functional code required by the extension is packaged within the extension itself.

7. Data security

We take reasonable steps to limit data access to what is necessary for the extension’s stated functionality. The extension is designed to process data locally wherever possible.

8. Changes to this Privacy Policy

We may update this Privacy Policy from time to time. Any updates will be reflected in the latest published version of this policy.
